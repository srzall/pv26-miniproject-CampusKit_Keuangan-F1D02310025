# 🎓 CampusKit - Pencatat Keuangan Mahasiswa

## 📖 Deskripsi Singkat
**CampusKit** adalah aplikasi desktop berbasis GUI (*Graphical User Interface*) yang dirancang khusus untuk membantu mahasiswa mencatat, mengelola, dan memantau arus keuangan harian (pemasukan dan pengeluaran). 

Aplikasi ini mendukung operasi CRUD (Create, Read, Update, Delete) secara penuh dan dilengkapi dengan *Dashboard* ringkasan saldo yang berjalan secara *real-time*. Proyek ini dibangun sebagai pemenuhan Tugas Mini Project Mata Kuliah Pemrograman Visual, dengan menerapkan prinsip arsitektur *Separation of Concerns (SoC)* agar kode lebih rapi, modular, dan mudah dikembangkan.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Python 3
* **GUI Framework:** PySide6
* **Database:** SQLite3 (Persisten)
* **Styling:** QSS (Qt Style Sheets) eksternal
